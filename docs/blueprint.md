# **App Name**: ResuAI

## Core Features:

- Resume Input: Allow users to input their resume via text area or .txt/.pdf file upload.
- AI Resume Summary: Use Google Gemini API to generate a concise summary of the resume content.
- Skill Extraction: Leverage the AI to extract key skills from the resume text.
- Weakness Detection: AI identifies potential weaknesses in the resume's content and formatting using Google Gemini as a tool.
- Improvement Suggestions: Generate tailored suggestions for improvements based on the weaknesses detected.
- Optimized Resume Generation: Create an optimized version of the resume based on AI analysis and suggestions using Google Gemini.
- Result History: Store and display the analysis results in local storage with the ability to clear the history.

## Style Guidelines:

- Primary color: Deep Indigo (#6666FF) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F5), providing a clean and neutral backdrop.
- Accent color: Teal (#40E0D0) to highlight important elements and call-to-actions, providing a sense of innovation.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, and neutral look suitable for both headlines and body text.
- Use clean, minimalist icons to represent different sections and actions.
- Dashboard layout with a left panel for resume input and a right panel for AI analysis results.
- Subtle, smooth animations for loading and content transitions to enhance user experience.
